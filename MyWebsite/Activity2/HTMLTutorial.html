What is HTML?

HTML(Hypertext Markup Language), is the standard markup language for web pages.

It consists of:

    Elements - Building blocks of HTML pages that have a start tag and end tag

Which can have

    Attributes - Details that provide additional information about the element

Part 1: Basic Syntax

The basic syntax of an HTML file looks like this:

<!DOCTYPE html> <html> <head> <title>[Type your title here]</title> </head> <body> <p>[Type text here]</p> </body> </html>
